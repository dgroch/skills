# Digital PR Pitch Email — Template

> Copy and modify per outlet. One unique angle per outlet — never send the same pitch to two outlets that compete for the same audience.

## Structural anatomy

**Subject line**
- Under 60 chars. Lead with the data/findings, never "Fig & Bloom announces…"
- A question or a surprising number works. "Only 6% of flower cards use emojis" beats "Fig & Bloom data report".

**Greeting**
- Always first name. Verify spelling via the journalist's byline / masthead.

**Hook (1–3 sentences)**
- Why this is news, why now, why THEM specifically. If you can't say why this journalist in the first sentence, you haven't earned the pitch yet. Tie to their beat.

**Key findings (3–5 bullets max)**
- Hard numbers only. This is what a sub-editor will screencap.

**Real anonymised excerpts (1–2, if emotional angle)**
- The "stop you in your tracks" quotes. Only for relationships / grief / lifestyle angles, not data desks.

**The offer**
```
What I can offer:
- Full anonymised raw dataset (CSV) — for data desks
- High-res images — [CONFIRMED WITH DANIEL FIRST]
- Interview with Kellie Brown, Fig & Bloom
- Exclusive (if offered)
```

**Link targets (gift-guide / listicle pitches only)**
```
Our flower delivery pages if useful for linking:
- Sydney: https://figandbloom.com/collections/flower-delivery-sydney
- Melbourne: https://figandbloom.com/collections/flower-delivery-melbourne
- Brisbane: https://figandbloom.com/collections/flower-delivery-brisbane
```

**Sign-off**
```
Best,
Dan Groch
Fig & Bloom
media@figandbloom.com
```

## Hard rules

- **No routine CC/BCC.** Daniel monitors `media@figandbloom.com` directly. Add him only when he explicitly requests a copy or a genuine escalation requires direct involvement.
- Plain text only. No HTML, no embedded images. Attachments via MEDIA: path if approved.
- No phone number in signature unless the journalist requests a call.
- One follow-up after 5 business days. No second follow-up — silence is an answer.
- Subject starts with "Re:" only when it's a genuine reply in an existing thread.

## Worked example: 6 angle variants

The flower-card campaign produced six pitch variants by angle — see `references/campaign-flower-card.md` for the full text of each:

1. **Data journalism** — leads with dataset offer + methodology
2. **Emotional / relationships** — leads with real card excerpts + love:sorry ratio
3. **Gift guide update** — leads with listicle refresh + collection links
4. **Grief / "no words"** — leads with the 23% sympathy/support figure
5. **Language / handwriting** — leads with "only 6% use emojis"
6. **Small business / innovation** — leads with "florist doing data journalism"
