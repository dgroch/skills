# Design Discipline

A craft-layer skill for **any visual creative** — social posts, stories,
carousels, banners, ads, posters, email headers, slide layouts, thumbnails.

Sits on top of a brand's own guide. The brand guide supplies the *what*
(fonts, palette, logo, voice); this skill supplies the *how* — the
grid, the type scale, the spacing rhythm, the critique loop that catches
the amateur mistakes before they ship.

## When to load

- Any time a layout is being composed, even if not explicitly asked to "design"
- When feedback says *"it looks off"*, *"it looks amateur"*, *"the spacing is
  wrong"*, *"the logo's too big"*, *"this doesn't feel like Fig & Bloom"*
- Before approving a render, banner, or static ad for any channel
- When bringing a new template or social format into the system

## What it enforces

- 6-column grid, 88px outer margins (1080 base)
- Modular type scale, max 2–3 sizes per view
- 8px spacing rhythm
- Exactly one focal element per view — logo is support, not hero
- Stories/Reels safe zones (top 14% / bottom 20%)
- Render at 100%, critique, iterate — never approve from a thumbnail
- Pre-ship QA checklist (12 items) before showing anyone
