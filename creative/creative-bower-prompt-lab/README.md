# Bower Prompt Lab

AI photography generator for Bower's Instagram feed. Uses Higgsfield Soul
for image generation and Claude Sonnet as an automated art-direction critic.

---

## What it does

You give it a brief (or just say "fill the grid"). It generates
brand-consistent product, lifestyle, and detail photography — scored
against Bower's art direction rubric — and builds a library of proven
prompts that improve over time.

**Three modes:**

| Mode | When to use | Example |
|------|-------------|---------|
| **Single shot** | Need one specific image | "Generate a product hero with peonies" |
| **Grid batch** | Fill the Instagram feed | "Generate 9 posts for the grid" |
| **Campaign set** | Seasonal or launch moment | "Mother's Day campaign — 5 shots" |

## Quick start

### 1. Set credentials

```bash
# OpenRouter (recommended — uses Nano Banana 2 for generation)
export OPENROUTER_API_KEY="sk-or-v1-..."

# Critic step: use EITHER direct Anthropic OR OpenRouter routing to Claude
# Option A — direct Anthropic API:
export ANTHROPIC_API_KEY="sk-ant-..."
# Option B — route critic via OpenRouter (only OPENROUTER_API_KEY needed):
# export CRITIC_MODEL="anthropic/claude-sonnet-4-5"  # or any OpenRouter Claude model

# Higgsfield (legacy — uses Soul)
# export HF_KEY="your-higgsfield-key"
# export HF_SECRET="your-higgsfield-secret"
# export ANTHROPIC_API_KEY="sk-ant-..."
```

> **OpenRouter-only setup:** If you set a `CRITIC_MODEL` in `anthropic/` namespace and have no
> `ANTHROPIC_API_KEY`, the critic will route through OpenRouter automatically. Only
> `OPENROUTER_API_KEY` is required in this configuration.

### 2. Run via CLI (interactive)

```bash
python3 bower_prompt_lab.py
```

Pick a shot number from the menu. The script generates, critiques,
and iterates until the image passes (score ≥ 7/10) or hits 5 attempts.

> Note: The CLI script still uses Higgsfield. For Nano Banana 2 via
> OpenRouter, use the Python API.

### 3. Run via Python API (for agents)

```python
# Add the references directory to Python path
import sys
sys.path.insert(0, "path/to/bower-prompt-lab/references")
from bower_api import BowerPromptLab

# OpenRouter + Nano Banana 2 (default)
lab = BowerPromptLab(
    openrouter_key="sk-or-v1-...",
    anthropic_key="sk-ant-...",
)

# Or: Higgsfield Soul (legacy)
# lab = BowerPromptLab(
#     backend="higgsfield",
#     hf_key="...", hf_secret="...",
#     anthropic_key="sk-ant-...",
# )

# Single shot
result = lab.generate("product_hero")
print(result["image_url"])   # Local file path or CDN URL
print(result["score"])       # e.g. 9

# Full grid
results = lab.generate_grid(season="spring")

# Campaign
results = lab.generate_campaign("mothers_day", flowers=["garden roses", "peonies"])

# Switch to Nano Banana Pro for higher quality (4x cost)
lab_pro = BowerPromptLab(
    openrouter_key="sk-or-v1-...",
    anthropic_key="sk-ant-...",
    model="google/gemini-3-pro-image-preview",
)
```

## How it works

```
Brief → Prompt Engine → Image Model → Claude Critic → Pass/Revise
                            ↑                            │
                            └──── iterate (up to 5x) ────┘
```

1. **Prompt Engine** selects or builds a prompt from the art direction rules.
2. **Image Model** generates the image:
   - **OpenRouter** (default): Nano Banana 2 — synchronous, ~10 seconds.
   - **Higgsfield** (legacy): Soul — async submit + poll, ~60–90 seconds.
3. **Claude Sonnet** scores it on 6 dimensions (lighting, composition,
   colour, styling, technical, brand alignment).
4. If score ≥ 7/10 → **pass**. Prompt locked and saved to library.
5. If score < 7 → Claude suggests revisions → revised prompt → try again.

## The Instagram grid

Bower's feed follows a 9-post pattern across 3 content lanes:

```
┌──────────────┬──────────────┬──────────────┐
│   Product    │  Lifestyle   │    Detail    │
│  close-up    │    wide      │   texture    │
├──────────────┼──────────────┼──────────────┤
│  Lifestyle   │   Product    │   Detail     │
│  doorstep    │    hero      │  packaging   │
├──────────────┼──────────────┼──────────────┤
│   Detail     │  Lifestyle   │   Product    │
│   petal      │    table     │    full      │
└──────────────┴──────────────┴──────────────┘
```

**Rules:**
- No more than 2 consecutive posts of the same type
- Every image shares the same warm colour grade
- Text posts: max 1 per 12 (not generated by this skill)
- At least 1 human element per 4 product shots

## The prompt library

Prompts are stored in `references/prompt-library.json`. Each entry has:
- The locked prompt text
- The critic's score and dimension breakdown
- The image URL (valid ~7 days from generation)
- Which iteration it passed on

The library grows every time you run the lab. Higher-scoring prompts
are preferred when the API wrapper selects from the library.

### Adding new prompts

Write prompts following this structure (all 10 components required):

1. **Shot type** — "Professional product photograph of..." / "Lifestyle photograph of..."
2. **Subject** — Specific flowers, vessel, arrangement style
3. **Camera** — Angle, format, lens
4. **Lighting** — Warm directional, colour temperature, shadow quality
5. **Depth of field** — f-stop, what's sharp vs blurred
6. **Composition** — Negative space %, framing
7. **Colour palette** — Use brand language (linen-cream, terracotta, sage-eucalyptus)
8. **Colour grade** — Warm shift, desaturated greens, lifted shadows
9. **Mood** — Emotional quality (tactile, warm, considered)
10. **Exclusions** — "No text, no logos, no watermarks"

### Creating variants

You don't need to write prompts from scratch. Use these swaps:

- **Flowers:** roses → peonies, dahlias, ranunculus, natives
- **Vessel:** terracotta vase → sandstone bowl → ribbed glass
- **Scene:** kitchen table → dining room → bedroom side table
- **Light:** morning → golden hour → overcast
- **Season:** add seasonal modifiers to mood and foliage

The Python API has built-in methods for these:

```python
lab.generate_campaign("spring", flowers=["peonies", "ranunculus"])
lab.generate_grid(season="autumn")
```

## Costs

**OpenRouter (default):**

| Model | Per generation (approx) | Best for |
|-------|------------------------|----------|
| Nano Banana 2 | ~$0.02–0.04 | Day-to-day feed content |
| Nano Banana Pro | ~$0.08–0.15 | Campaign hero shots |

**Higgsfield (legacy):** ~4–5 credits per generation on Soul.

**Claude critic:** ~1K tokens per critique call (~$0.003 per call on Sonnet).

Typical cost per passing image: 1–2 generations + 1–2 critiques.
Worst case (5 iterations): 5 generations + 5 critiques + 4 revision calls.

## Limitations

- Prompts were originally tested against Higgsfield Soul — scores may
  vary on Nano Banana 2. Run the 6 verified prompts through to recalibrate.
- OpenRouter returns images as base64 — they're saved locally, not to a CDN
- AI-generated hands and human elements are the weakest area across
  all models — expect occasional artefacts in lifestyle shots
- The critic can be lenient on first attempts — consider raising
  threshold to 8 for higher quality
