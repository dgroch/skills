# Fig & Bloom — Occasion Defaults

Pre-mapped defaults by campaign occasion. Load this file at the start of every intake.

Apply defaults silently, then surface them for confirmation — don't ask cold for information that can already be inferred.

When `opt_out_required: true`, always explicitly surface this for human confirmation before finalising the brief. Never set it silently.

---

## Occasion map

### Mother's Day
- **opt_out_required:** TRUE ← confirm with user
- **Primary persona:** Milestone Sender
- **Secondary persona:** Considered Giver
- **Accent colour default:** Blush / soft pink / dusty rose (`#E8C4C4` or similar)
- **Tone notes:** Warm, sentimental but never saccharine. Acknowledge the occasion is not universal.
- **Shopify handle suggestion:** `mothers-day` or `gifts`
- **Notes:** Highest-sensitivity campaign of the year. Opt-out segment is essential. Sympathy/loss angle warrants a separate quiet email variant if volume justifies it.

### Valentine's Day
- **opt_out_required:** TRUE ← confirm with user
- **Primary persona:** Milestone Sender, Considered Giver
- **Secondary persona:** Everyday Aesthete (self-purchase angle)
- **Accent colour default:** Deep red, blush, or warm white — avoid clichéd red unless intentional
- **Tone notes:** Don't over-romance it. Considered > sentimental. Self-purchase angle worth including.
- **Shopify handle suggestion:** `valentines` or `gifts`
- **Notes:** Sensitivity to relationship loss / grief. Opt-out segment required.

### Sympathy / Grief / Condolence
- **opt_out_required:** TRUE ← confirm with user
- **Primary persona:** Milestone Sender
- **Secondary persona:** None
- **Accent colour default:** None — brand defaults only (avoid warm/celebratory palettes)
- **Tone notes:** Restrained, quiet, unhurried. No promotional language. No offer.
- **Shopify handle suggestion:** `sympathy` or `condolence`
- **Notes:** No promotional offer should appear in sympathy campaigns. Tone is the product here.

### Easter
- **opt_out_required:** FALSE
- **Primary persona:** Everyday Aesthete
- **Secondary persona:** Entertainer
- **Accent colour default:** Warm sage, soft yellow, or terracotta
- **Tone notes:** Seasonal freshness. Home/table angle. Not religious-specific unless brief directs.
- **Shopify handle suggestion:** `easter` or `spring`

### Christmas / End of Year
- **opt_out_required:** FALSE (confirm — bereavement sensitivity may still apply depending on list)
- **Primary persona:** Considered Giver
- **Secondary persona:** Milestone Sender
- **Accent colour default:** Deep green, warm white, or dried/neutral — avoid supermarket red/green clichés
- **Tone notes:** Generous, unhurried. Premium gifting angle. Avoid "Christmas sale" energy.
- **Shopify handle suggestion:** `christmas` or `gifts`

### Spring / Seasonal (general)
- **opt_out_required:** FALSE
- **Primary persona:** Everyday Aesthete
- **Secondary persona:** Entertainer
- **Accent colour default:** Ask user, or leave as brand defaults
- **Tone notes:** Editorial, seasonal. Nature-led language. Native flower focus where applicable.
- **Shopify handle suggestion:** `spring` or `seasonal`

### Birthday (gifting campaign)
- **opt_out_required:** FALSE
- **Primary persona:** Milestone Sender
- **Secondary persona:** Considered Giver
- **Accent colour default:** None — brand defaults
- **Tone notes:** Warm but not generic. Avoid "Happy Birthday" banner energy.
- **Shopify handle suggestion:** `birthday` or `gifts`

### Subscription Promo / Evergreen
- **opt_out_required:** FALSE
- **Primary persona:** Everyday Aesthete
- **Secondary persona:** Trade Buyer
- **Accent colour default:** Brand defaults
- **Tone notes:** Living well. Consistency and ritual. Not discount-led unless brief says otherwise.
- **Shopify handle suggestion:** `subscription` or `fresh-flowers`

### New Baby / Congratulations
- **opt_out_required:** FALSE
- **Primary persona:** Milestone Sender
- **Secondary persona:** Considered Giver
- **Accent colour default:** Soft white, sage, or warm neutral
- **Tone notes:** Celebratory but restrained. Avoid baby-pink clichés unless requested.
- **Shopify handle suggestion:** `gifts` or `congratulations`

### New Year / January Reset
- **opt_out_required:** FALSE
- **Primary persona:** Everyday Aesthete
- **Secondary persona:** Milestone Sender
- **Accent colour default:** Cool white, pale green, or minimal
- **Tone notes:** Fresh start, home reset energy. Subscription angle natural here.
- **Shopify handle suggestion:** `subscription` or `fresh-flowers`

### B2B / Trade Outreach
- **opt_out_required:** FALSE
- **Primary persona:** Trade Buyer
- **Secondary persona:** None
- **Accent colour default:** Brand defaults — keep it clean
- **Tone notes:** Professional, reliability-focused. Not content-led. Testimonials and specifics over lifestyle imagery.
- **Shopify handle suggestion:** N/A — brief should specify custom product/service offering

### Ad Hoc / No Occasion
- **opt_out_required:** FALSE
- **Primary persona:** Ask user
- **Secondary persona:** Ask user
- **Accent colour default:** Brand defaults
- **Tone notes:** Ask user for campaign-specific direction

## Brand voice quick-reference

Always active, regardless of occasion:

- **Tone:** Warm, editorial, minimal, premium, Australian English
- **Avoid these words:** elevate, curate, stunning, gorgeous, pop of colour
- **Prefer:** Considered, intentional, seasonal, natural, understated
- **Register:** Conversational but never casual. Confident but never loud.
