# Publisher Benchmark — what "editorial" actually means

Study **structure and craft**, never copy a publisher's words or images. The patterns
below are what separates a publisher carousel from a brand template.

## 1 · Cover / hook
- One photograph + one line (**≤ 8 words**) + a quiet masthead. The image is the hook.
- Open a **curiosity gap** — name the subject, withhold the answer. Don't let the cover
  resolve itself.
- The masthead is *support*, never the loudest thing on the cover.
- Break the headline by **sense**; never strand a single word.
- *Seen in:* Vogue Living, Gourmet Traveller, The Design Files (stylist-art-directed covers).

## 2 · Type hierarchy
- **Cereal:** hierarchy from *minimal* size variation — most of the magazine in one serif
  at one size; ranking done by **negative space**. Drama from air, not point size.
- **Kinfolk:** a serif + a humanist sans, three colours, restrained scale; obsessive about
  **no widows, no rivers, no ugly hyphenation**.
- Two type roles held all the way through. No third font, no surprise weights.
- *Take:* a tight ~3-step ladder; hierarchy from scale + space + colour. Emphasis is rare
  and never faked — emphasis uses the brand's real italic face (Cervanttis), not a synth slant.

## 3 · Pacing
- A clear **arc**: open → develop → turn → resolve.
- **Alternate texture** (photo / type / photo). Two heavy type slides back-to-back is the
  most common amateur tell.
- **Consistency *with* rhythm:** keep one system, change **one accent per slide**. That
  repetition-with-variation reads as "a designed report".
- **Open loops** between slides pull the swipe. 7–10 slides; curated, not inflated.

## 4 · Image-to-text ratio
- Lifestyle publishers run **image-forward**; even an essay is paced with imagery.
- Photography is **treated, not just placed**: consistent grade, a horizontal hero crop,
  generous margins, type in the image's quiet zone with real scrim/contrast.
- Full-bleed verticals used sparingly; **never two portraits stacked** — pair as a band.
- *Take:* ≈ 40% image moments, woven not clustered. Text-forward is fine if the text
  slides vary.

## 5 · Caption ↔ carousel
- **Slides = headlines; caption = the article.** Long-form prose, asides and context live
  in the caption, not on the slides.
- First caption line **complements** slide 1 (deepens the hook), never echoes it.
- Closer slide + caption CTA are **one move**.

## 6 · How they close
- A **sign-off, not a sell:** masthead, byline, one quiet pointer onward.
- The closer **resolves the arc** — it pays off the loop the cover opened.
- No discounting, no urgency — the brand's authority is the call to action.

## House standard (distilled)

| Dimension | Bar |
|---|---|
| Format | 4:5 — 1080×1350 design / 2160×2700 export |
| Cover | 1 photo · 1 line ≤ 8 words · quiet masthead · opens a loop |
| Type roles | display serif + grotesk only · ~3 sizes total |
| Emphasis | rare; Cervanttis italic / colour / line-break — never a synth slant on the display face |
| Arc | open → develop → turn → resolve; no two same-texture slides adjacent |
| Image moments | ≈ 40%, woven; horizontal heroes; never stacked portraits |
| Body copy | short on-slide; prose in the caption |
| Negative space | composed and anchored — a void with no base anchor reads unfinished |
| Close | POV sign-off + author + masthead; no discount/urgency |
