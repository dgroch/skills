# Worked Example — "I Only Ever Use One Foliage" critique

A real run of `critique-rubric.md` against a finished 9-slide deck (Kellie's column on
floral restraint). Use it as the model for tone, specificity, and how to rank fixes —
the goal is named gaps and concrete fixes, never "could be stronger".

## Highest-value gaps (ranked)

1. **Straight quotes throughout** (tell #5). `isn't` / `It's` un-curled on slides 2, 3, 5, 9;
   glaring at Lust display size. Cheapest fix, biggest lift — replace with `'` `"`.
2. **Dead emphasis on slide 5** (tell #1). *"edited object,"* the turn phrase of the whole
   deck, renders flat — the original build leaned on a roman face for an emphasis it couldn't
   carry, so the `<em>` did nothing. Fix: set it in **Cervanttis italic** (the brand's real
   italic display face), used once.
3. **Type-slide repetition** (tell #2). Only two interior templates (clay-statement,
   noir-statement) reused; slides 5-6-7 form a three-slide type run; 2→3 are adjacent with
   identical layout. Fix: a breath slide + varied vertical placement.
4. **Unanchored bases** (tell #7). Slides 2, 3, 5, 6, 7 float with empty lower thirds; only
   the closer is anchored. Fix: a quiet running foot on text slides.
5. **Folio contrast on photos** (tell #8). `04` and `08` folios vanish on bright sky/petals.
   Fix: a top scrim or per-background colour.

## POV / honesty flags

- **Slide 8 — proof contradicts the claim (Honesty = 2).** Headline reads "One focal flower.
  One foliage. One palette," but the arrangement visibly uses three foliages (dusty miller,
  eucalyptus, hypericum berries). The six craft axes all pass; only Honesty catches it.
  Fix: recrop/reshoot to a frame that shows a single foliage, or soften the claim.
- **Slide 4 — provenance unverified.** The English cottage garden isn't an owned asset;
  legitimate as licensed scene-setting, but confirm the licence before publishing.

## What this run proved about the skill

- It **bites**: every "template tell" it lists was actually present and findable, and the
  scoring guide landed the deck exactly where it belonged (3.6 = brand template).
- It **would have prevented** the slide-5 failure — that gap exists *because* the build
  predated the skill's emphasis rules.
- The Honesty axis exists *because of* slide 8: in an earlier six-axis version the
  foliage-vs-claim mismatch slipped through every axis and was caught only by the closing
  POV-flag line. Now it has a home.
